
# Atlas

Atlas is an AI-powered workplace simulation platform that helps professionals prepare for real-world jobs through interactive simulations.

## Overview

Instead of reading theory, Atlas allows users to practice workplace scenarios, make decisions, receive AI feedback, and build confidence before entering a role.

## Key Features

- AI-generated workplace simulations
- Interactive decision making
- Personalized feedback
- Progress tracking
- Industry-specific learning paths

## Target Users

- Students
- Graduates
- Career changers
- Professionals preparing for interviews
- Corporate training teams

## Technology

- Lovable
- React
- TypeScript
- Supabase
- OpenAI APIs
- GitHub

## Status

Current stage: MVP (Minimum Viable Product)

The platform is under active development with continuous feature releases.
